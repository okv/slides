Постановка задачи

Задача обуечния по прецедентам

Дано:

X — множество объектов

Y — множество выходов

Y*: X -> Y — неизвестная зависимость (target function)

X = {x | x <- x1..xn} — обучающая выборка

yi = y*(xi), i = 1..n — известные ответы

Найти:

a: X -> Y — алгоритм, решающая функция, приближающая y* на всем множестве X

Объекты и признаки

fj: X -> Dj, j = 1..n — признаки объектов

Признаки бывают: бинарными, номинальными, порядковыми, количественными.

Вектор (f1(x), f2(x), ..., fn(x)) — признаковое описание объекта x

Из векторов признаков составляется матрица F — матрица «объекты-признаки»

Постановка задачи

Необходимо разбить запросы к серверу на три класса: Положительные запросы (запросы от пользователей), Запросы от ботов, Запросы генерируемые ботнетом.

Примеры признаков в нашей задаче

  • Наличие или отсутствие User-Agent;
  • Территориальная принадлежность отправителя запроса;
  • Параметры запроса;
  • Ожидание ответа;
  • Количество запросов с одного IP-адреса.
